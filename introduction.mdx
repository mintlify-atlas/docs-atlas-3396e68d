## What is ZenHogar?

ZenHogar is a collaborative household task management platform designed for roommates, families, Erasmus students, and anyone sharing a living space. Built with Angular and Firebase, it brings harmony to shared living by making task organization transparent, fair, and engaging.

## Key Features

ZenHogar offers a comprehensive set of features to streamline household task management:

- **Google Authentication with Firebase** - Quick and secure login with your Google account
- **Multi-household support** - Manage multiple living spaces from one account
- **Task creation and assignment** - Create tasks with custom descriptions and weights
- **Points-based rating system** - Fair evaluation with peer ratings and point tracking
- **Email-based invitations** - Invite members via email with unique access codes
- **Real-time notifications** - Stay updated on task assignments and completions
- **Task request workflows** - Request task reassignments with approval system
- **Historical tracking** - View complete task history and contributions
- **Admin controls** - Transfer ownership and manage household settings

## Technology Stack

ZenHogar is built with modern, production-ready technologies:

- **Frontend**: Angular 20 with TypeScript
- **UI Framework**: Angular Material with custom SCSS styling
- **Backend**: Firebase (Authentication + Firestore)
- **Email Service**: EmailJS for invitations and notifications
- **Real-time Updates**: RxJS observables for reactive data flow
