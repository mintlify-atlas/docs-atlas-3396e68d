Welcome to ZenHogar! This guide will walk you through setting up your account, creating your first household, inviting members, and creating your first task.

## Prerequisites

Before you begin, make sure you have:
- A Google account for authentication
- Access to the ZenHogar application (deployed or running locally)

## Get Started in 4 Steps

<Steps>
  <Step title="Sign in with Google">
    ZenHogar uses Google authentication for secure, hassle-free login.

    1. Navigate to the ZenHogar application
    2. Click the **Sign in with Google** button
    3. Select your Google account and authorize the application

    <Note>
      ZenHogar only requests basic profile information (name, email, and photo) to personalize your experience.
    </Note>

    The authentication service will automatically create your user profile in Firestore:

    ```typescript src/app/core/auth/auth.service.ts
    async loginGoogle() {
      const { user } = await firebaseAuthWrapper.signInWithPopup(
        this.injector,
        this.auth,
        new GoogleAuthProvider()
      );

      // Create or update user document
      await firestoreWrapper.setDoc(ref, {
        nombre: user.displayName ?? '',
        displayName: user.displayName ?? '',
        email: (user.email ?? '').toLowerCase(),
        photoURL: user.photoURL ?? '',
        lastLogin: serverTimestamp(),
        hogarId: null,
        createdAt: serverTimestamp(),
      }, { merge: true });
    }
    ```
  </Step>

  <Step title="Create Your First Household">
    After signing in, you'll need to create a household to start managing tasks.

    1. Click on **Create Household** or the **+** button
    2. Fill in the household details:
       - **Name**: Give your household a memorable name (e.g., "Apartment 4B", "Smith Family")
       - **Type**: Choose from Familiar, Pareja, Amigos, or Erasmus
       - **Country**: Select your country
       - **Province/Region**: Select your province or region
    3. Click **Create**

    <CodeGroup>
    ```typescript Household Model
    export interface Hogar {
      id?: string;
      nombre: string;
      countryCode: string;
      provincia: string;
      provinciaCode: string;
      ownerUid: string;
      miembros: string[];
      createdAt: Timestamp | null;
      tipoHogar?: 'Familiar' | 'Pareja' | 'Amigos' | 'Erasmus';
      zenPrime?: boolean;
    }
    ```

    ```typescript Creating a Household
    async crearHogar(
      nombre: string,
      provincia: string,
      user: User,
      tipoHogar?: TipoHogar
    ): Promise<{ id: string }> {
      const docDataHogar: Omit<Hogar, 'id'> = {
        nombre: nombre.trim(),
        countryCode: cc,
        provincia,
        provinciaCode: code,
        ownerUid: user.uid,
        miembros: [user.uid],
        createdAt: serverTimestamp(),
        tipoHogar: tipoHogar ?? 'Familiar',
        zenPrime: false,
      };

      const ref = await addDoc(collection(this.fs, 'hogares'), docDataHogar);
      
      // Create default tasks for the household
      await this.tareasSvc.crearTareasPorDefecto(ref.id, user.uid);
      
      return { id: ref.id };
    }
    ```
    </CodeGroup>

    <Note>
      When you create a household, ZenHogar automatically generates a set of default tasks to help you get started!
    </Note>
  </Step>

  <Step title="Invite Household Members">
    Now that your household is created, invite your roommates, family members, or friends to join.

    1. Click on the **Invite Member** button
    2. Enter the email address of the person you want to invite
    3. Click **Send Invitation**
    4. Share the generated invitation code with them

    The invited person will:
    1. Sign in with their Google account (using the same email)
    2. Click **Join Household**
    3. Enter the invitation code
    4. Accept to join your household

    <CodeGroup>
    ```typescript Create Invitation
    async crearInvitacion(hogarId: string, email: string): Promise<string> {
      const codigo = uuidv4().slice(0, 8);

      await setDoc(doc(this.fs, 'invitaciones', codigo), {
        hogarId,
        email: email.toLowerCase(),
        codigo,
        creadoEn: serverTimestamp(),
        usado: false,
      });

      return codigo;
    }
    ```

    ```typescript Accept Invitation
    async aceptarCodigo(codigo: string): Promise<void> {
      const invit = await this.validarCodigo(codigo);
      if (!invit) throw new Error('Código inválido o ya usado');

      await runTransaction(this.fs, async (trx) => {
        // Add member to household
        trx.set(memberRef, {
          uid: user.uid,
          codigo,
          email,
          joinedAt: serverTimestamp(),
        });

        // Mark invitation as used
        trx.update(invRef, {
          usado: true,
          usadoPorUid: user.uid,
          usadoEn: serverTimestamp(),
        });

        // Update user's household reference
        trx.set(userRef, {
          hogarId,
          totalTareasRealizadasHogar: 0,
        }, { merge: true });
      });
    }
    ```
    </CodeGroup>

    <Note>
      Invitation codes are unique 8-character codes tied to a specific email address and can only be used once.
    </Note>
  </Step>

  <Step title="Create and Assign Your First Task">
    With your household set up and members invited, you're ready to create tasks!

    ### Create a Task

    1. Click **New Task** or the **+** button in the tasks section
    2. Fill in the task details:
       - **Name**: What needs to be done (e.g., "Take out trash", "Clean kitchen")
       - **Description**: Additional details (optional)
       - **Weight**: Task difficulty/importance (1-5)
    3. Click **Create Task**

    ### Assign a Task

    You can assign tasks in multiple ways:

    - **Self-assign**: Click on an unassigned task and select yourself
    - **Assign to member**: Select a household member from the dropdown
    - **Request assignment**: Send a task request to another member

    <CodeGroup>
    ```typescript Create Task
    async crearTarea(data: {
      nombre: string;
      descripcion?: string;
      hogarId: string;
      peso?: number;
      creadorUid: string;
    }): Promise<string> {
      const payload: Omit<Tarea, 'id'> = {
        nombre: data.nombre,
        descripcion: data.descripcion ?? '',
        completada: false,
        asignadA: null,
        hogarId: data.hogarId,
        createdAt: serverTimestamp(),
        peso: data.peso,
        historial: [],
        valoraciones: [],
        valoracionesPendientes: [],
        bloqueadaHastaValoracion: false,
        personalizada: true,
        creadorUid: data.creadorUid,
      };

      const ref = await addDoc(colRef, payload);
      return ref.id;
    }
    ```

    ```typescript Assign Task
    async asignarTarea(
      tareaId: string,
      nuevoUid: string
    ): Promise<void> {
      const tareaRef = doc(this.fs, 'tareas', tareaId);
      const usuarioRef = doc(this.fs, 'usuarios', nuevoUid);
      const snap = await getDoc(usuarioRef);
      
      const user = snap.data() as Usuario;

      await updateDoc(tareaRef, {
        asignadA: nuevoUid,
        asignadoNombre: user.nombre,
        asignadoFotoURL: user.photoURL || null,
      });
    }
    ```

    ```typescript Request Assignment
    async crearPeticionAsignacion(data: {
      tareaId: string;
      hogarId: string;
      deUid: string;

    }
    ```
    </CodeGroup>
  </Step>
</Steps>

## Task Completion Flow

Once a task is assigned, here's how the completion workflow works:

1. **Complete the task**: Mark the task as complete when finished
2. **Request ratings**: Other household members receive a notification to rate the task
3. **Members rate**: Each member rates the task completion (1-5 stars)
4. **Points awarded**: The person who completed the task receives points based on the average rating and task weight

```typescript Task Rating System
// Points are calculated based on rating and task weight
const PUNTOS_POR_VALORACION = {
  1: 10,   // Poor
  2: 25,   // Fair
  3: 50,   // Good
  4: 75,   // Very Good
  5: 100   // Excellent
};

const puntosBase = PUNTOS_POR_VALORACION[media];
const puntosOtorgados = puntosBase * taskWeight;
```
