## Overview

The `TareasService` is a comprehensive service for managing household tasks. It handles task creation, assignment, completion, ratings, task requests between members, and points calculation. It integrates closely with the home and user systems.

**Location**: `src/app/core/tareas/services/tareas.service.ts`

**Injectable**: `{ providedIn: 'root' }`

<ParamField path="getTareasPorHogar(hogarId, enrich?)" type="Observable<TareaDTO[]>">

</ParamField>

**Parameters**:
<Expandable title="Parameters">

```typescript
  });
});
```

## Task Assignment

### asignarTarea()

<ParamField path="asignarTarea(tareaId, nuevoUid, penalizacionAbandono?)" type="Promise<void>">
  Assigns or unassigns a task. Records assignment history and optionally applies abandonment penalty.
</ParamField>

**Parameters**:
<Expandable title="Parameters">
  <ResponseField name="tareaId" type="string" required>
    ID of the task
  </ResponseField>
  <ResponseField name="nuevoUid" type="string" required>
    UID of new assignee, or empty string to unassign
  </ResponseField>
  <ResponseField name="penalizacionAbandono" type="number" default="-5">
    Penalty points when unassigning an in-progress task (negative number)
  </ResponseField>
</Expandable>

**Throws**:
- `'Modo demo: no se puede asignar tarea.'`: No authenticated user
- `'Tarea no encontrada'`: Task doesn't exist
- `'Solo quien la tiene en curso puede reasignar.'`: Task in progress by another user
- `'Usuario no encontrado'`: Target user doesn't exist

**Behavior**:
- Adds previous assignment to task history
- Applies penalty if unassigning in-progress task
- Updates assignee name and photo

**Example**:
```typescript
// Assign task to user
await this.tareasService.asignarTarea(taskId, userId);

// Unassign task (with penalty if in progress)
await this.tareasService.asignarTarea(taskId, '', -10);

// Unassign without penalty
await this.tareasService.asignarTarea(taskId, '', 0);
```

## Task Completion & Rating

### finalizarTareaConCancelacion()

<ParamField path="finalizarTareaConCancelacion(tarea, usuarioActual, miembros)" type="Promise<void>">
  Marks a task as complete and triggers the rating workflow if there are other members to rate it.
</ParamField>

**Parameters**:
<Expandable title="Parameters">
  <ResponseField name="tarea" type="TareaDTO" required>
    Task to complete
  </ResponseField>
  <ResponseField name="usuarioActual" type="object" required>
    Current user object with `uid`, `displayName`, `photoURL`
  </ResponseField>
  <ResponseField name="miembros" type="{ uid: string }[]" required>
    Array of home members
  </ResponseField>
</Expandable>

**Behavior**:
- **Case 1**: Other members exist → Sets `completada: true`, adds to history, sets `valoracionesPendientes` array, unassigns
- **Case 2**: No other members → Closes task without points, marks as `sin_valoradores`, leaves unassigned

**Example**:
```typescript
const user = this.auth.currentUser;
const miembros = [{ uid: 'user1' }, { uid: 'user2' }, { uid: 'user3' }];

await this.tareasService.finalizarTareaConCancelacion(
  tarea,
  {
    uid: user.uid,
    displayName: user.displayName,
    photoURL: user.photoURL
  },
  miembros
);

console.log('Task completed, awaiting ratings');
```

### valorarTarea()

<ParamField path="valorarTarea(tareaId, puntuacion, comentario, uidActual)" type="Promise<void>">
  Submits a rating for a completed task. When all ratings are submitted, awards points to the task completer.
</ParamField>

**Parameters**:
<Expandable title="Parameters">
  <ResponseField name="tareaId" type="string" required>
    ID of the task
  </ResponseField>
  <ResponseField name="puntuacion" type="number" required>
    Rating value (typically 1-5)
  </ResponseField>
  <ResponseField name="comentario" type="string" required>
    Rating comment
  </ResponseField>
  <ResponseField name="uidActual" type="string" required>
    UID of the user submitting the rating
  </ResponseField>
</Expandable>

**Throws**:
- `'Modo demo: no se puede valorar tarea.'`: No authenticated user
- `'Tarea no encontrada'`: Task doesn't exist
- `'Ya has valorado esta tarea.'`: User already rated this task
- `'No tienes esta tarea pendiente de valorar.'`: User not in pending raters list

**Behavior**:
- Adds rating to `valoraciones` array
- Removes user from `valoracionesPendientes`
- When all ratings submitted:
  - Calculates average rating
  - Converts to points using `PUNTOS_POR_VALORACION` mapping
  - Multiplies by task weight
  - Awards points to task completer
  - Updates user's `totalTareasRealizadas` and `totalTareasRealizadasHogar`
  - Unassigns task (ready for next assignment)
  - Sets `completada: false`

**Example**:
```typescript
try {
  await this.tareasService.valorarTarea(
    taskId,
    5,  // Excellent rating
    '¡Excelente trabajo!',
    currentUserId
  );
  console.log('Rating submitted');
} catch (error) {
  if (error.message === 'Ya has valorado esta tarea.') {
    alert('You already rated this task');
  }
}
```

## Complete Example

```typescript
import { Component, OnInit } from '@angular/core';
import { TareasService } from '@core/tareas/services/tareas.service';
import { AuthService } from '@core/auth/auth.service';

@Component({
  selector: 'app-tasks',
  template: `
    <div *ngFor="let tarea of tareas$ | async">
      <h3>{{ tarea.nombre }}</h3>
      <p>{{ tarea.descripcion }}</p>
      <p>Assigned to: {{ tarea.asignadoNombre || 'Unassigned' }}</p>
      <p>Weight: {{ tarea.peso }}x</p>
      
      <button *ngIf="!tarea.asignadA" 
              (click)="assignToMe(tarea.id!)">
        Assign to me
      </button>
      
      <button *ngIf="tarea.asignadA === currentUid" 
              (click)="complete(tarea)">
        Mark complete
      </button>
      
      <button *ngIf="tarea.completada && needsMyRating(tarea)" 
              (click)="rate(tarea.id!, 5, 'Great job!')">
        Rate task
      </button>
    </div>
    
    <h3>My Points This Week: {{ puntosSemana$ | async }}</h3>
  `
})
export class TasksComponent implements OnInit {
  tareas$ = this.tareasService.getTareasPorHogar(this.hogarId);
  puntosSemana$ = this.tareasService.getPuntosSemana(
    this.hogarId,
    this.currentUid
  );
  
  hogarId = 'home-123';
  currentUid = this.authService.uidActual!;

  constructor(
    private tareasService: TareasService,
    private authService: AuthService
  ) {}

  ngOnInit() {
    this.tareas$.subscribe(tareas => {
      console.log('Tasks loaded:', tareas.length);
    });
  }

  async assignToMe(tareaId: string) {
    await this.tareasService.asignarTarea(tareaId, this.currentUid);
  }

  async complete(tarea: any) {
    const user = this.authService.currentUser!;
    const miembros = [{ uid: 'user1' }, { uid: 'user2' }];
    
    await this.tareasService.finalizarTareaConCancelacion(
      tarea,
      {
        uid: user.uid,
        displayName: user.displayName,
        photoURL: user.photoURL
      },
      miembros
    );
  }

  async rate(tareaId: string, rating: number, comment: string) {
    await this.tareasService.valorarTarea(
      tareaId,
      rating,
      comment,
      this.currentUid
    );
  }

  needsMyRating(tarea: any): boolean {
    return tarea.valoracionesPendientes?.includes(this.currentUid) ?? false;
  }
}
```
